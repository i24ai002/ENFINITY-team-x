## 🧠 Problem

Financial apps today are overloaded with:

* Charts
* Graphs
* Notifications

👉 Users feel confused instead of informed.

---

## 💡 Solution

**$ GULLAK** gives **one clear answer:**

> 💰 *How much can I safely spend today?*

No clutter. No noise. Just clarity.

---

## 🎯 Core Feature

**Safe Daily Spend (30-Day Runway)**

Safe Daily Spend = (Balance - Recurring Expenses) / 30

---

## ✨ Features

* 📥 Upload CSV / Excel (Bank + PaySim)
* 🧠 Backend Financial Engine (Next.js API)
* 💰 Safe Daily Spend Calculation
* ⚠️ Anomaly Detection
* 🔍 Semantic Search
* 🎨 Ultra Minimal UI (No charts)

---

## 🏗 Tech Stack

* Frontend: Next.js, Tailwind CSS
* Backend: Next.js API Routes
* Parsing: XLSX + Custom Logic
* Deployment: Vercel
